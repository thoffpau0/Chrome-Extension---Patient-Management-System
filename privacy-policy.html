Privacy Policy — Vet Radar Notification

Effective Date: June 7, 2025  |  Version: 1.6

1. Introduction

Vet Radar Notification ("the Extension") is a Chrome extension that monitors the VetRadar patient board and plays audio alerts when patient or task events occur. This policy describes what data the Extension touches, how it is handled, and what is never collected.

2. Data the Extension Reads

The Extension reads the VetRadar page DOM in real time to detect changes such as patients being added or removed and tasks being assigned or completed. This reading happens entirely within your browser. No patient data — including names, species, breed, medical conditions, or any clinical information — is ever stored, transmitted, or logged by the Extension. Data read from the DOM is held only in the JavaScript memory of the active tab and discarded when the page is navigated away or the tab is closed.

3. Data the Extension Stores

The Extension stores only your personal settings using Chrome's storage.local API. This includes:

  • Sound selections and volume levels for each alert channel
  • Enable/disable toggles per channel
  • Master volume level
  • Custom audio file data (if you upload your own MP3), stored as a Base64 data URL
  • Whether monitoring was active when the extension was last used

This data is stored locally on your device only. It is never transmitted to any server.

4. Data We Do Not Collect

  • No patient names, medical records, or clinical data
  • No usage analytics or telemetry
  • No browsing history
  • No personally identifiable information
  • No cookies or tracking technologies

5. Third-Party Services

The Extension does not communicate with any external server or third-party service. All processing occurs locally within Chrome on your device.

6. Permissions Used

  • storage — to save your sound and volume preferences locally
  • host_permissions (app.vetradar.com) — to inject the monitoring script into the VetRadar patient board page

7. Data Retention and Deletion

Settings are retained in chrome.storage.local until you uninstall the Extension or use the "Restore All Defaults" button in the Options page. Uninstalling the Extension removes all stored settings automatically.

8. User Rights

You can view, reset, or delete all Extension settings at any time via the Options page (chrome://extensions → Vet Radar Notification → Details → Extension options). No account or request to us is required.

9. Changes to This Policy

If this policy is updated, the new version will be posted in the Extension's GitHub repository and the effective date above will be revised.
